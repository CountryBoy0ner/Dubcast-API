## Elevator Pitch

This project is a web-based radio platform that provides a shared real-time music stream for all listeners. Users can start listening instantly without registration, while authorized users can interact through a built-in live chat and personalized profiles. The platform addresses the problem of choice overload in modern music services by offering passive, continuous listening without playlist management. Its uniqueness lies in the use of human-curated playlists instead of algorithm-driven recommendations, creating a more organic and community-oriented listening experience.
